# 5. Ограничения

## 5.1 Технические ограничения

### TECH-1: Архитектура
- Одностраничное приложение (SPA)
- Полное отсутствие серверной части
- Хостинг на GitHub Pages

**Ссылки**: [[02_overview](02_overview.md)

---

### TECH-2: Выполнение кода
- Python через Pyodide (WebAssembly)
- Локальные `.whl` файлы для зависимостей
- Ограничения памяти браузера

**Ссылки**: [[04_interfaces](04_interfaces.md#API-1)

---

### TECH-3: Работа с файлами
- Чтение IDS — через File API браузера
- Запись IFC — через Blob/Download API
- Все операции локально, без сети

---

## 5.2 Зависимости
| Компонент | Версия | Назначение |
|-----------|--------|------------|
| Pyodide | v0.29.3 | Python 3.13 runtime (совместим с ifcopenshell wheel) |
| ifcopenshell | 0.8.4 (cp313, pyodide_2025_0) | IFC API |
| IDS parser | встроенный | Парсинг IDS XML |

**Важно:** Версия Pyodide зафиксирована на v0.29.3 из-за совместимости wheel ifcopenshell.
Использование `dev` версии (Python 3.14+) или v0.27.2 (Python 3.12) приведёт к ошибке установки.
