# 9a. Сценарии: Скачивание и логи

## UC-4: Скачивание результата

**Актор**: BIM-координатор

**Предусловие**: IFC сгенерирован

**Основной поток**:
1. Пользователь нажимает «Скачать IFC»
2. Браузер загружает файл `IDS2PSET_Library.ifc`

**Постусловие**: Файл сохранён локально

**Ссылки**: [[03_features](03_features.md#FR-6), [[08a_result_mockups](08a_result_mockups.md)

---

## UC-5: Просмотр логов

**Актор**: Любой пользователь

**Предусловие**: Выполняется или завершён процесс

**Основной поток**:
1. Пользователь открывает панель логов
2. Система отображает пошаговый лог с временными метками
3. Пользователь копирует лог (опционально)

**Ссылки**: [[03_features](03_features.md#FR-5), [[08a_result_mockups](08a_result_mockups.md)
