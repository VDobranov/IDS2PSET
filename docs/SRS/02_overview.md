# 2. Общее описание

## 2.1 Продуктовая перспектива
Новый продукт, развивающий идеи проекта [ANCHOR-BOLT-GENERATOR](https://github.com/VDobranov/ANCHOR-BOLT-GENERATOR).

## 2.2 Классы пользователей
| Роль | Описание |
|------|----------|
| **BIM-координатор** | Контролирует соответствие моделей требованиям IDS |
| **Инженер-проектировщик** | Создаёт проектные модели, нуждается в шаблонах свойств |
| **Разработчик САПР** | Интегрирует генерацию PSet в свои инструменты |

## 2.3 Предположения и зависимости
- Пользователь имеет валидный файл IDS
- Браузер поддерживает WebAssembly (Pyodide)
- Локальное выполнение кода (без сервера)

## 2.4 Диаграмма контекста
```
┌─────────────┐    IDS-файл    ┌─────────────┐

                               └─────────────┘
```
